# Changelog

All notable changes to this project will be documented here.

## [0.2.0] - 2025-12-31

### Added

- Selectable WCAG contrast thresholds (3.0 / 4.5 / 7.0)
- Dynamic filtering of the color wheel based on selected level
- (additional) Font change
**Full Changelog**: [at this link](https://github.com/giansteve/contrast-color-wheel/compare/v0.1.0...v0.2.0)

### Fixed

- Contrast filtering logic now applies the strictest selected threshold

## [0.1.0] - 2025-12-29

### Added

- Initial release: WCAG contrast wheel with isolines, drag selector, live preview.
